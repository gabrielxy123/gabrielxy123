## Typing Effects
```javascript
const typingEffect = (element, text) => {
  let index = 0;
  const interval = setInterval(() => {
    if (index < text.length) {
      element.textContent += text.charAt(index);
      index++;
    } else {
      clearInterval(interval);
    }
  }, 100);
};
```
